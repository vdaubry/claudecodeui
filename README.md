# Claude Code UI

Web interface for [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code). Access your projects and conversations from desktop or mobile.

## Quick Start

```bash
npx @siteboon/claude-code-ui
```

Opens at `http://localhost:3001`

## Requirements

- Node.js v20+
- [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code) installed

## Development

```bash
git clone https://github.com/siteboon/claudecodeui.git
cd claudecodeui
npm install
npm run dev
```
